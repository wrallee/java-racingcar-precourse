# 자동차 경주 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 목록
- [X] 자동차 이름을 입력받는다.
  - [X] 자동차 이름 길이 확인
  - [ ] 자동차 이름 중복 확인
- [ ] 0~9 사이의 랜덤 값을 생성한다.
  - [X] 생성된 값을 검증한다.
- [ ] 4 이상이면 자동차가 전진하고 3 이하면 움직이지 않는다.
- [ ] 자동차의 위치를 출력한다.
- [ ] 최대 이동거리인 차들을 출력한다.
